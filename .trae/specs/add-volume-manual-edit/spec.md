# 分卷框架手动填写 Spec

## Why
作品概览页面的分卷框架区域当前只能展示AI生成的内容，没有手动添加和编辑的功能。当AI未生成分卷框架或用户想自行调整时，需要支持手动添加、编辑和删除分卷。

## What Changes
- 在分卷框架区域添加"添加分卷"按钮，支持手动新增分卷
- 每个分卷条目添加编辑和删除按钮
- 编辑模式下可修改卷名、描述、预计章节数、核心冲突
- 保存时将分卷框架数据写入 worldApi（title 为 "volumes"）

## Impact
- Affected code: `src/pages/NovelDetail.tsx`（分卷框架区域）

## ADDED Requirements

### Requirement: 分卷框架手动填写
系统 SHALL 在作品概览页面的分卷框架区域提供手动添加、编辑和删除分卷的功能。

#### Scenario: 手动添加分卷
- **WHEN** 用户点击"添加分卷"按钮
- **THEN** 出现一个新的分卷编辑表单，包含卷名、描述、预计章节数、核心冲突字段
- **THEN** 用户填写后点击保存，分卷数据保存到 worldApi

#### Scenario: 编辑已有分卷
- **WHEN** 用户点击某个分卷的编辑按钮
- **THEN** 该分卷变为可编辑状态，用户可修改各字段
- **THEN** 保存后更新 worldApi 中的数据

#### Scenario: 删除分卷
- **WHEN** 用户点击某个分卷的删除按钮
- **THEN** 确认后删除该分卷，保存更新后的数据
